# GTM Insights Assistant 

An AI-powered tool that answers natural language questions about sales pipeline data.

## Tech Stack
- Python, Pandas
- OpenRouter API (LLM)
- Prompt Engineering

## How to Run
1. pip install pandas requests
2. Add your OpenRouter API key in main.py
3. python main.py

## Example Questions
- Which region has the highest deal value?
- Which product performs best?
- Which sales rep has the most deals?
